docker常用命令

Jun 29 2019
1.拉取镜像
docker pull centos
2.查看已经下载的镜像
docker images  

  REPOSITORY          TAG                 IMAGE ID            CREATED             SIZE
  centos              latest              49f7960eb7e4        6 weeks ago         200MB
3.运行一个镜像并生成容器
docker run centos
4.查看运行中的镜像
docker ps  
CONTAINER ID        IMAGE               COMMAND             CREATED             STATUS              PORTS               NAMES

docker container ls  
CONTAINER ID        IMAGE               COMMAND             CREATED             STATUS              PORTS               NAMES
5.提交镜像到镜像仓库
docker login  
docker tag imagename username/imagename:tag  
docker psuh username/imagename:tag
6.宿主机容器端口号映射并运行
docker run -d -p 80:8080 --name mynginx nginx
7.DockerFile创建镜像
docker build -t imagenamge:tag .

-t ：指定镜像名称和标签，格式为’name:tag’ .: 最后一个点代表当前目录，也可以换成其它的路径

8.删除本地镜像
docker rmi IMAGE ID

-f 强行移除该镜像，即使其正被使用

9.删除容器
docker rm CONTAINER ID

-f 强行移除该镜像，即使其正被使用

10.Maven中生成镜像
mvn clean package docker:build
