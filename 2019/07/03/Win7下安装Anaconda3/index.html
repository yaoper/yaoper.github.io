Win7下安装Anaconda3

Jul 03 2019
1.下载安装包

官网安装包下载地址 安装包下载地址，Win下注意区分要下载的是32bit还是64bit，一路Next完成安装。

2.运行 jupyter notebook

在CMD命令行输入 jupyter notebook, 默认浏览器会打一个地址http://localhost:8888/tree/
jupyter notebook运行成功

3.可能遇到的问题
  • a.jupyter’不是内部或外部命令，也不是可运行的程序或批处理文件
    Path里面配置
    C:\Users\Administrator\Anaconda3\Scripts;
    C:\Users\Administrator\Anaconda3;
    C:\Users\Administrator\Anaconda3\Library\bin
4.更换jupyter工作空间

jupyter默认的工作空间是C:\Users\Administrator,修改默认的工作空间要打开C:\Users\Administrator.jupyter找到jupyter_notebook_config.py文件修改c.NotebookApp.notebook_dir = ‘(自己的路径)’，工作空间目录要提前创建
如果找不到jupyter_notebook_config.py这个文件，可使用jupyter notebook –generate-config命令生成

5.关于网页无法访问的问题

命令行启动jupyter能正常打开浏览器访问，但是你可能打不开其他网页了，百度新浪都不行。可尝试重启电脑后重试（不一定有用）。
