Dlib在WIN下的安装记录

Jul 12 2019

1.dlib简介

dlib 是一个机器学习的开源工具包（官网：http://dlib.net/ ），用于机器人、嵌入式设备、移动电话和大型高性能计算环境，也可以做人脸检测识别等；官网给出的Python Examples:

  • Binary Classification
  • CNN Face Detector
  • Face Alignment
  • Face Clustering
  • Face Detector
  • Face Jittering/Augmentation
  • Face Landmark Detection
  • Face Recognition
  • Find Candidate Object Locations
  • Global Optimization
  • Linear Assignment Problems
  • Sequence Segmenter
  • Structural Support Vector Machines
  • SVM-Rank
  • Train Object Detector
  • Train Shape Predictor
  • Video Object Tracking

dlib的安装在WIN下不是很友好，这里推荐在Anaconda环境下安装，这样就省去了build 和 make dlib 的源码，这样安装会更顺利。

2.安装包下载

  • Anaconda3-5.1.0
  • dlib
  • pip

强烈建议按照上述地址下载，因为Anacon3-5.1.0需要与dlib的版本匹配，后面会讲版本不匹配的问题。如上述下载地址不能下载可访问清华大学开源软件镜像站、Git个人站自行下载。
