Python 安装第三方库慢，没速度的解决方法

Jul 12 2019

刚刚接触Python，在使用pip install安装Python第三方库的过程中会出现网速很慢，或者是安装下载到中途停止、报错等问题。
这类问题是因网络原因导致访问Python源不可达造成的，解决办法是更换Python源。

1.临时解决办法

在安装pip install -i 源地址 安装库名，这样可以临时解决安装问题。
eg: pip install -i https://pypi.tuna.tsinghua.edu.cn/simple opencv-python
