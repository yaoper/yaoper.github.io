可能遇到需要安装Docker Compose
Step0:首先检查linux有没有安装python-pip包，直接执行 yum install python-pip
Step1:没有python-pip包就执行命令 yum -y install epel-release
Step2:执行成功之后，再次执行 yum install python-pip
Step3:对安装好的pip进行升级 pip install –upgrade pip
Step4:安装Docker Compose，执行pip install -U docker-compose
Step5:检验docker-compose是否安装成功,执行docker-compose –version
