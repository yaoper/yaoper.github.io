neo4j安装教程

Aug 06 2019

1.安装包下载

neo4j提供Unix/Mac/Win平台的安装包，这里我们下载Unix平台 neo4j-community-3.5.8-unix.tar.gz

2.依赖环境准备

neo4j使用Java语言开发，所以需要安装Java环境，这里推荐至少安装Java 8及其以上。

3.修改配置文件

配置环境变量, vi /etc/profile

vi /etc/security/limits.conf

配置允许远程WEB访问， vin /usr/local/neo4j/conf/neo4j.conf,打开这些配置

启动neo4j, cd /usr/local/neo4j/conf/

4.设置账号密码

wen登录ip:7474,使用账号密码neo4j/neo4j登录成功后，修改密码。

5.安装完成检测

cd /usr/local/neo4j/bin

./cypher-shell

CREATE (A {id:1,name:’张飞’}), (B {id:2,name:’张小二’}), (C {id:3,name:’张三’}), (D {id:4,name:’张凯’}),(E {id:5,name:’张雨’}),(A)-[:Son]->(B),(A)-[:Son]->(C),(B)-[:Son]->(D),(C)-[:Son]->(E);

match (n) return n;
